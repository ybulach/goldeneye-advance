Une image vide de 8x8 pixels est présente dans interface/interface.bmp.
Cette image sert de support et tout les autres sprites doivent y être insérés.

Ordre d'insertion:
	1/ barre_tir
	2/ barre_vies
	3/ curseur
	4/ interface
	5/ font
	6/ james_bond
	7/ scientifiques

Il ne doit y avoir que 250 tiles maximum pour que les sprites de l'interface puissent être utilisés aussi bien en mode 3 à 5 qu'en mode 0 à 2.
La taille maximale doit donc être de 16 Ko.

La palette doit être de 256 couleurs.
